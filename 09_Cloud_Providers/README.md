# 09. Cloud Providers

Di era modern, sebagian besar infrastruktur berjalan di Cloud. Anda perlu memahami konsep Cloud Computing.

## Konsep Cloud
1.  **IaaS (Infrastructure as a Service)**: AWS EC2, Google Compute Engine.
2.  **PaaS (Platform as a Service)**: Heroku, Google App Engine, AWS Elastic Beanstalk.
3.  **SaaS (Software as a Service)**: Gmail, Salesforce.
4.  **FaaS (Function as a Service)**: AWS Lambda, Google Cloud Functions (Serverless).

## Provider Utama
1.  **AWS (Amazon Web Services)**
    *   Pemain terbesar.
    *   Service kunci: EC2 (VM), S3 (Storage), RDS (Database), VPC (Network), IAM (Security), Lambda (Serverless).
2.  **Google Cloud Platform (GCP)**
    *   Kuat di Data & AI, serta Kubernetes (GKE).
3.  **Microsoft Azure**
    *   Integrasi kuat dengan ekosistem Microsoft/Enterprise.

## Latihan Praktis
1.  Buat akun Free Tier di AWS/GCP/Azure.
2.  Launch sebuah Virtual Machine, install web server, dan akses dari public internet.
3.  Setup S3 bucket untuk menyimpan file statis.
4.  Pelajari konsep IAM (Identity and Access Management) dengan membuat user yang hanya memiliki akses read-only ke S3.
