# DevOps Roadmap & Learning Materials

Selamat datang di panduan pembelajaran DevOps. Folder ini disusun berdasarkan roadmap standar industri untuk membantu Anda menjadi DevOps Engineer yang handal.

## Struktur Pembelajaran
Silakan ikuti materi ini secara berurutan:

1.  [**Bahasa Pemrograman**](./01_Programming_Language/README.md)
    *   Pelajari Python atau Go sebagai fondasi scripting dan automation.
2.  [**Sistem Operasi**](./02_Operating_Systems/README.md)
    *   Pahami Linux, manajemen proses, memori, dan file system.
3.  [**Terminal & Scripting**](./03_Terminal_and_Scripting/README.md)
    *   Kuasai Bash, Vim, dan command line tools.
4.  [**Networking & Security**](./04_Networking_and_Security/README.md)
    *   Pelajari TCP/IP, DNS, HTTP, SSL/TLS, dan Firewall.
5.  [**Web Servers**](./05_Web_Servers/README.md)
    *   Konfigurasi Nginx, Apache, dan konsep Reverse Proxy.
6.  [**Infrastructure as Code (IaC)**](./06_Infrastructure_as_Code/README.md)
    *   Docker, Kubernetes, Terraform, dan Ansible.
7.  [**CI/CD**](./07_CI_CD/README.md)
    *   Otomatisasi build dan deploy dengan Jenkins, GitHub Actions, atau GitLab CI.
8.  [**Monitoring & Logging**](./08_Monitoring_and_Logging/README.md)
    *   Prometheus, Grafana, dan ELK Stack.
9.  [**Cloud Providers**](./09_Cloud_Providers/README.md)
    *   Pengenalan AWS, Google Cloud, atau Azure.
10. [**Design Patterns**](./10_Design_Patterns/README.md)
    *   Microservices, Serverless, dan 12-Factor App.
11. [**Capstone Project**](./11_Capstone_Project/README.md)
    *   Proyek akhir membangun pipeline DevOps lengkap dari nol.
12. [**Career & Interviews**](./12_Career_and_Interviews/INTERVIEW_QUESTIONS.md)
    *   Kumpulan pertanyaan interview dan tips karir.
13. [**Troubleshooting Guide**](./13_Troubleshooting_Guide/REAL_WORLD_SCENARIOS.md)
    *   Skenario masalah dunia nyata dan cara menyelesaikannya.

## Tips Belajar
*   **Praktik adalah Kunci**: Jangan hanya membaca. Cobalah setiap latihan yang diberikan.
*   **Fokus Satu Hal**: Jangan mencoba mempelajari semuanya sekaligus. DevOps sangat luas.
*   **Dokumentasikan**: Catat apa yang Anda pelajari. Itu akan menjadi referensi berharga nantinya.

Selamat belajar!
